\doxysubsection{Detailed Description}
Represents a person in the simulation. 

The {\ttfamily \doxylink{class_person}{Person}} class simulates a person\textquotesingle{}s daily routine, including their activities such as going to work, going home, and going to their favorite building (e.\+g., a recreational spot). The class handles scheduling these activities based on random distributions for timings, and simulates the movement between buildings. It manages the person\textquotesingle{}s transportation, which involves driving a car to different destinations.

Attributes include the person\textquotesingle{}s home, work, and favorite building, as well as the current action and state (e.\+g., driving or not). The class updates the person\textquotesingle{}s current action depending on the time of day and performs the corresponding action (e.\+g., going to work, returning home).
