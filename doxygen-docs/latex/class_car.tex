\doxysubsection{Detailed Description}
Represents a car in the simulation. 

The {\ttfamily \doxylink{class_car}{Car}} class simulates a car\textquotesingle{}s behavior, including movement, direction, and interactions with roads and intersections. It supports functionality to control the car\textquotesingle{}s position, speed, and state while driving towards a destination. The class handles transitions between roads and intersections, updates car position based on its speed, and allows interaction with other cars.
