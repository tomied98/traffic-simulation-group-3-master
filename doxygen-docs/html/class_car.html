Detailed Description

Represents a car in the simulation.

The Car class simulates a car's behavior, including movement, direction, and interactions with roads and intersections. It supports functionality to control the car's position, speed, and state while driving towards a destination. The class handles transitions between roads and intersections, updates car position based on its speed, and allows interaction with other cars.
