Detailed Description

Represents a bus in the simulation that follows a route with multiple stops.

The Bus class simulates a bus's behavior, including its movement along a route with predefined stops. It extends the Car class and supports the functionality to visit the next stop in its route. The bus moves at a defined speed and waits at each stop before moving to the next. The stops are cycled through in a loop, ensuring the bus continuously follows its route. The bus's behavior also includes a waiting period at each stop before continuing.

void Bus::visitNextStop ( )

Visits the next stop in the bus's route.

The bus drives to the next stop, waits for a period (controlled by waitTime_), and then updates the current stop index. The bus loops through the stops cyclically.

< The bus waits for 80 units of time after reaching a stop.

< Move to the next stop, looping if necessary.

< Decrement the wait time while the bus is waiting at the stop.
